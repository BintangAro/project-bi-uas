A. Struktur Proyek
- `airflow_etl/dags/` → Pipeline ETL dengan Airflow untuk 3 Star Schema
- `django_dashboard/` → Dashboard visualisasi hasil analisis dengan Django + Chart.js
- `data_input/` → Dataset dimensi dan fakta (fact)
- `data_output/` → Output hasil transformasi

B. Star Schema yang Digunakan
1. Penjualan Berdasarkan Wilayah
2. Analisis Model 5G dengan Revenue Tertinggi
3. Kontribusi Wilayah terhadap Total Penjualan
